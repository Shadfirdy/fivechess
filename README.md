
### Feature 

- 悔棋

- 重新開局

- 終端介面紀錄座標，方便debug

### How to start

```shell
$ cd ~

$ git clone https://github.com/yiyu0x/fivechess

$ cd fivechess

$ javac FiveGame.java

$ java FiveGame

```
### Tips

開局由黑棋先下，不斷輪流。

按下重新開局之後，即可直接下載新的位置。

### Bugs

重新開局之後畫面不會馬上清空，必須等到下一手。
